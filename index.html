Hardening para Servidores

Un script fácil, configurable que nos ayudara a proteger nuestros servidores
Ver en GitHub Descargar .zip Descargar .tar.gz


flash_on

Fácil y Rápido

La necesidad de tener todo en un solo script es tener las configuraciones a la mano un ejemplo es configurar ssh de manera correcta, bloquear puertos innecesarios, reglas de iptables para bloquear peticiones, entre otros.

group

Usuarios con experiencias en Linux

Es importante destacar la participación de los usuarios con experiencia en linux que estén dedicados a la seguridad informática, desarrolladores dedicados a la programación segura y sysadmin.

settings

Fácil de configurar

La configuración del script es muy sencillo, la usabilidad es importante a la hora de hacer una configuración en nuestro servidores. Con el tiempo podremos añadir herramientas de seguridad importantes en el script como por ejemplo ossec, etc...
